Live Tiles support in IE11

Scenario 3: Using XML

Back to scenarios menu

Description

The following meta tag points the page to an XML file which contains your pinned site markup. The XML file (IEconfig.xml) will set the color of your tile, the various tile sizes and the tile notifications.

To see this sample code in action, pin this page to the Windows 8.1 Start screen from IE11 (in the new Windows UI).

In Page markup

<meta name="msapplication-config" content="IEconfig.xml" />

IEconfig.xml

<?xml version="1.0" encoding="utf-8"?>
<browserconfig>
  <msapplication>
    <tile>
      <square70x70logo src="images/smalltile.png"/>
      <square150x150logo src="images/mediumtile.png"/>
      <wide310x150logo src="images/widetile.png"/>
      <square310x310logo src="images/largetile.png"/>
      <TileColor>#009900</TileColor>
    </tile>
    <notification>
      <polling-uri  src="notifications/contoso1.xml"/>
      <polling-uri2 src="notifications/contoso2.xml"/>
      <polling-uri3 src="notifications/contoso3.xml"/>
      <frequency>30</frequency>
      <cycle>1</cycle>
    </notification>
  </msapplication>
</browserconfig>
