Live Tiles support in IE11

Scenario 1: Using HTML

Back to scenarios menu

Description

The following HTML markup will set the name of your pinned site, the tile color, the various tile sizes and the tile notifications.

  • The application-name tag sets the default name of your pinned site
  • The TileColor tag sets the colour for your pinned site. This colour is used as the solid background colour for your live tile and is also used to calculate the text colour.
  • The four square and wide image tags set the four tile sizes for this tile. Note that the image you set for the medium tile size will also be used as the tile background when your site is a Favorite or Frequent site.
  • The notification tag sets the polling URIs to be polled, and the frequency with which they are pulled. The contents of the polling xmls are linked below. This code sample does not demonstrate updated notifications, but in a typical live-tile implementation, a script will regularly update the xml files with fresh notifications.

To see this sample code in action, pin this page to the Windows 8.1 Start screen from IE11 (in the new Windows UI).

Code

  <meta name="application-name" content=" Contoso" />
  <meta name="msapplication-TileColor" content=" #009900" />
  <meta name="msapplication-square70x70logo" content="images/smalltile.png" />
  <meta name="msapplication-square150x150logo" content="images/mediumtile.png" />
  <meta name="msapplication-wide310x150logo" content="images/widetile.png" />
  <meta name="msapplication-square310x310logo" content="images/largetile.png" />
  <meta name="msapplication-notification" content="frequency=30; polling-uri=notifications/contoso1.xml;
    polling-uri2=notifications/contoso2.xml; polling-uri3=notifications/contoso3.xml" />
